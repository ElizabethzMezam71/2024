### 虚拟信用卡使用注意事项

Fomepay 本质上是一种虚拟借记卡，开卡后余额为 0，如果不充值是无法扣费使用的，所有需要在充值后再进行订阅/支付。

为了成功绑定 OpenAI 或开通 ChatGPT Plus 会员，可以使用 Fomepay 提供的人工客服。切记未充值不要去绑定，因为绑定信用卡时平台会进行扣费验证，如果余额为零会造成绑卡失败和增加消费失败率。

Fomepay 的虚拟卡不需要外网，使用国内网络打开即可。
